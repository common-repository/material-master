The management of material information (Name, Size, Weight, Unit, Photo).

== Description ==
It can used as the material master. 

Management fields are as follows.

* "Type"
* "Name"
* "Unit"
* "Size"
* "Weight"
* "Photos"
* "Notes"
* "Date"

Before you can use this plug-in, please Install and Activate ["Advanced Custom Fields"](https://wordpress.org/plugins/advanced-custom-fields/) plugin.

= How to use? =

1. Installation plugins.
1. Login to WordPress. Select `Material Master` and `Add New` in Admin screen.
1. Write Material informations. And Publish.
1. Next, Select `Posts` and `Add New`.
1. Write shortcode `[Material-List]` at editor-area, And Publish.

= Shortcods =

#### [Material-List]

This shortcode will list the Materials. 



== Installation ==

1. *FIRST* Install ["Advanced Custom Fields"](https://wordpress.org/plugins/advanced-custom-fields/) and Activation.
1. Upload the entire `material-master` folder to the /wp-content/plugins/ directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
